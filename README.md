# Memory Cleaner – очистка системного кэша Windows

## Описание

**Memory Cleaner** — это простое десктопное приложение для быстрого удаления временных файлов Windows. Программа очищает все папки `TEMP` текущего пользователя, освобождая дисковое пространство от ненужного кэша. Приложение работает в многопоточном режиме, отображая прогресс обработки файлов и подробную статистику.

Для доступа к системной папке `C:\Windows\Temp` программа запрашивает права администратора (при необходимости перезапускается с повышенными привилегиями). Файлы, которые в данный момент используются системой или другими приложениями, пропускаются без ошибок.

## Возможности

- **Автоматический сбор** временных файлов из двух основных папок Windows
- **Удаление файлов** с учётом занятых (пропускаются без прерывания)
- **Удаление пустых папок** после очистки файлов
- **Прогресс-бар** и подробная статистика:
  - количество обработанных / удалённых / пропущенных файлов
  - объём удалённой и оставшейся памяти
  - имя текущего обрабатываемого файла
- **Запрос прав администратора** при необходимости (с возможностью отказа)
- **Работа в фоновом потоке** – интерфейс не зависает

## Как это работает

Программа сканирует указанные директории, рекурсивно собирая все файлы и подсчитывая их общий размер. Затем последовательно удаляет каждый файл, предварительно снимая атрибут «Только для чтения» (если требуется). В процессе удаления обновляется прогресс, счётчики и имя текущего файла. После завершения удаления файлов программа находит и удаляет все пустые папки внутри очищаемых директорий.

## Требования для .py-скрипта

- Python 3.7 или новее
- Стандартная библиотека Python (дополнительные пакеты не требуются)

## Установка и запуск

- Клонируйте репозиторий
- (Если запускать из .py-файла) Установите зависимости из requirements.txt и запустите .py-скрипт.
- (Если запускать из .exe-файла) Запустите .exe-программу.

## Инструкция по использованию

1) **Запустите программу**
   – Если вы не обладаете правами администратора, появится запрос на перезапуск с повышенными привилегиями (можно отказаться, но тогда системный кэш `C:\Windows\Temp` очищен не будет).

2) **Нажмите «Начать очистку»**
   – Программа соберёт все файлы из папок:
     - `C:\Users\Имя_пользователя\AppData\Local\Temp`
     - `C:\Windows\Temp`

3) **Наблюдайте за процессом**
   – Отображается прогресс-бар, процент выполнения, количество удалённых/пропущенных файлов и освобождённое место.

4) **Дождитесь завершения**
   – По окончании появится окно с итоговой статистикой. Все временные файлы и пустые папки будут удалены.

## Примечания

- Файлы, которые в момент очистки используются другим процессом, будут пропущены – это нормальное поведение Windows.
- Удаление происходит без возможности восстановления (не через Корзину). Будьте внимательны, но временные папки по определению содержат только ненужные данные.
- Для полной очистки `C:\Windows\Temp` требуются права администратора. Без них программа удалит только пользовательские временные файлы.
- Программа не удаляет системные или важные файлы – она работает только внутри указанных временных директорий.
